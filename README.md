# C++  ELzero Bootcamp

This repository contains Solutions for C++ bootcamp.

## Structure

The project is organized into weekly modules:

- `control-flow-week5`: Exercises related to control flow (if/else, switch case).
- `array-week6`: Exercises related to arrays.
- `loops-week7`: Exercises related to loops.

Each folder contains C++ source files (`.cpp`) for various tasks.

## How to Run

To compile and run the programs, use a C++ compiler like `g++`:

```bash
g++ task-1.cpp -o task-1
./task-1
```

## Requirements

- C++ Compiler (GCC, Clang, MSVC)
